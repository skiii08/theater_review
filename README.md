Theater Review App：映画館のスクリーンや座席からの見え方をレビューできる独自機能を持った映画レビュー共有プラットフォーム
概要
一般的な映画レビューサイトとは異なり、映画館のレビューサイトと映画レビューサイトを融合したサービス。
映画の内容だけでなく、どの座席からどのように見えるかという「視聴体験」もレビューできる独自の機能を提供。

主要機能

・実装済み機能

ユーザー認証システム（Laravel Breeze）
映画検索・表示（TMDB API連携）
映画レビュー機能
映画館・スクリーン情報管理
座席からの見え方レビュー（独自機能）
投稿機能
画像アップロード（Cloudinary連携）

・開発予定機能

フォロー機能
レビューのいいね機能
推薦システムとの統合

技術スタック

・バックエンド

PHP 8.1+
Laravel 10
MySQL

・フロントエンド

Blade Templates
Tailwind CSS
JavaScript

・外部サービス

TMDB API（映画データ取得）
Cloudinary（画像管理）

・認証・セキュリティ

Laravel Breeze（認証システム）
Laravel Sanctum（API認証）

データベース設計

・主要なテーブル：

users - ユーザー情報
movies - 映画情報（TMDB API連携）
theaters - 映画館情報
screen_numbers - スクリーン情報
reviews - 映画レビュー
theater_reviews - 座席・視聴体験レビュー
posts - 投稿情報

特徴
・座席レビューシステム

スクリーン番号と座席位置を指定したレビュー
見やすさ、音響、画面の見え方などを詳細評価
映画館選びの新しい指標を提供

・映画館データベース

映画館とスクリーンの詳細情報管理
座席配置や設備情報の蓄積

開発のポイント
・データベース設計

正規化を意識したテーブル設計
映画、劇場、スクリーン、レビューの関連性をモデリング

・Laravel の実践

MVC設計パターンの実装
Eloquent ORM によるデータベース操作
API連携（TMDB）
画像アップロード機能

・外部サービス連携

TMDB APIによる映画データ取得
Cloudinaryによる画像管理
